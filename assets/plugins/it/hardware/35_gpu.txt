<h1>La carte graphique</h1>



<h2>Une calculatrice surpuissante</h2>

<h3>Gérer l'affichage graphique</h3>
<p>
Le <a href="https://fr.wikipedia.org/wiki/Processeur_graphique" target="_blank">processeur graphique</a>, aussi appelé GPU, permet à l'ordinateur de générer une image pouvant être affiché sur un écran.
Il peut se présenter en général sous plusieurs formes différentes : sous forme de carte graphique (extension du système), soit sous forme d'un ajout au processeur, soit sous forme d'une puce intégrée à la carte mère (IGP)...
Il s'agit probablement d'un des composants les plus complexes d'un ordinateur.En effet, <span class="important">pour générer une image satisfaisante, il doit réaliser plein de tâches, toutes aussi complexes les unes que les autres : géométrie 2D / 3D, rendu d'image, rendu vidéo...</span>
En général, ces tâches sont accélérées via la configuration matériel du processeur graphique, le rendant extrêmement performant dans le domaine.
De plus, on peut même moduler le processeur graphique pour lui faire faire tout ce que l'on veut (grâce à, notamment, la librairie "CUDA" sur les processeurs graphiques NVidia).
</p>
