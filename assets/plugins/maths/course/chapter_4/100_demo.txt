<h1>Chapitre 4 - démonstrations</h1>



<h2>Les anneaux et les corps</h2>

<h3 id="polynomial">Les polynômes</h3>
<h4>4.0.0.0</h4>
<p>
Démontrons que dire qu'un polynôme "P(X)" admet une racine "r" équivaut à dire que ce polynôme est divisible par "X - r".
Procédons par double implication.
</p>
<p>
Prouvons que dire qu'un polynôme "P(X)" est divisible par "X - r" implique que "P(X)" admet une racine "r".
Si "P" est divisible par "X - r", il existe un "Q(X)" tel que le produit de "Q(X)" et "X - r" donne "P".
</p>
<math><mexists> Q <min> K[X], (X - r) * Q(X) = P(X)<math>
<p>
De manière évidente, "r" permet d'annuler "(X - r)".
Par transitivité de l'égalité, "r" annule aussi "P(X)".
</p>
<math>X = r <mimp> (X - r) = 0 <mimp> (X - r) * Q(X) = P(X) = 0</math>
<p>
Donc, dire qu'un polynôme "P(X)" est divisible par "X - r" implique que "P" admet une racine "r".
</p>
<p>
Prouvons qu'un polynôme "P(X)" admet une racine "r" implique que "P(X)" est divisible par "X - r".
Pour cela, nous devons juste prouver l'existence d'un polynôme "Q" tel que "X - r" multiplié par "Q" donne "P".
Cela implique que la division euclidienne de "P(X)" par "X - r" (qui existe bel et bien) donne un reste nul.
</p>
<math>P(X) = Q(X) * (X - r) + R(X)</math>
<p>
Nous savons qu'une division euclidienne de polynômes est unique.
Donc, la division euclidienne de ce polynôme pour "X = r" est exactement la même que pour tous "X".
Or, si "X = r", "P(X) = 0" et "Q(X) * (X - r) = 0" : il n'y a pas besoin de reste, qui est bel et bien nul.
</p>
<math>X = 0 <mimp> (P(X) = Q(X) * (X - r) + R(X) <mand> R(X) = 0 <mequi> P(X) = Q(X) * (X - r)) <mimp> R(X) = 0</math>
<p>
Donc, un polynôme "P(X)" admet une racine "r" implique que "P(X)" est divisible par "X - r".
</p>
<p>
Les deux implications sont vérifiées : dire qu'un polynôme "P(X)" admet une racine "r" équivaut à dire que ce polynôme est divisible par "X - r".
</p>



<h2>Les espaces vectoriels</h2>

<h3>Les applications linéaires</h3>

<h4>4.1.0.0</h4>
<p>
Démontrons que, pour tout "w" et "z" appartenant à "R", l'application allant de "R²" vers "R²" transformant "(x, y)" en "(w * x, z * y)" est linéaire.
Déjà, la première propriété est respectée, et l'on peut le vérifier de manière algébrique.
</p>
<math><math><mforall>a, b <min> <mi>R</mi><msup>2</msup>, <mexists>x, x', y, y' <min> R, a = (x, y), b = (x', y'), f(a + b) = f(x + x', y + y') = (w * (x + x'), z * (y + y')) = (w * x + w * x', z * y + z * y') = (w * x, z * y) + (w * x', z * y') = f(a) + f(b)</math>
<p>
La deuxième propriété est aussi vraie, et ce n'est pas si difficile que ça à démontrer non plus.
</p>
<math><math><mforall>a, b <min> <mi>R</mi><msup>2</msup>, <mexists>x, y, c <min> R, a = (x, y), f(c * a) = f(c * x, c * y) = (w * c * x, z * c * y) = c * (w * x, z * y) = c * f(a)</math>
<p>
Finalement, "f(0, 0) = (0, 0)", donc cette application est belle est bien linéaire.
Donc, pour tout "w" et "z" appartenant à "R", l'application allant de "R²" vers "R²" transformant "(x, y)" en "(w * x, z * y)" est linéaire.
</p>
