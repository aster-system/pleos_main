<h1>Chapitre 2 - démonstrations</h1>



<h2>Introduction à l'algèbre</h2>

<h3>Propriétés basiques des lois de compositions</h3>

<h4>2.0.0.0</h4>
<p>
Démontrons que, si une loi commutative "." permet l'existence un élément neutre dans un ensemble E, alors il est unique.
</p>
<p>
Rappelons la définition d'un élément neutre "e" :
</p>
<math><mforall>a <min> E, a . e = a</math>
<p>Définissons un élément "f" obéissant aussi à cette propriété :</p>
<math><mforall>a <min> E, a . f = a</math>
<p>Donc :</p>
<math>e . f = e</math></br><math> f . e = f</math>
<p>Par commutativité :</p>
<math>e . f = f . e = e = f</math>
<p>Tout élément obéissant à cette propriété est "e" : il est donc unique.</p>

<h3>Les structures algébriques de base</h3>

<h4>2.0.1.0</h4>
<p>
Démontrons que la structure (N, +) est un magma (et donc, une structure algébrique).
Pour cela, nous avons besoin que d'une seule preuve : "+" est stable sur "N".
Or, nous l'avons démontré, <a href="demo-1#1_2_1_0">dans le chapitre 1, proposition 1.2.1.0</a>.
Donc, (N, +) est bien une structure algébrique.
</p>

<h4>2.0.1.1</h4>
<p>
Démontrons que, pour tout "n" appartenant à "N" supérieur à 1, "nZ" est un sous-groupe (additif) du groupe additif "(Z, +)".
Déjà, vérifions si "nZ" est stable sur l'addition.
</p>
<math><mforall>a, b <min> nZ, <mexists> c, d, e <min> Z, [a = n * c, b = n * d, a + b = e]</math></br>
<math><mimp>e = a + b = n * c + n * d = n * (c + d)</math></br>
<math><mimp><mexists> f <min> Z, [f = c + d, e = n * f] <mimp> e <min> nZ</math>
<p>
Comme nous le constatons, toute somme d'éléments de "nZ" forme un élément de "nZ".
Maintenant, vérifions la présence de chaque opposé de chaque élément de "nZ".
</p>
<math><mforall>a <min> nZ, <mexists> b, c <min> Z, [a = n * b, a + c = 0]</math></br>
<math><mimp>n * b + c = 0 <mimp> n * b = -c <mimp> n * -b = c <mimp> c <min> nZ</math>
<p>
Chaque élément possède bien un opposé dans "nZ".
Toutes les conditions sont remplies, les groupes "nZ" sont des sous-groupes du groupe additif "(Z, +)".
</p>

<h4>2.0.1.2</h4>
<p>
Démontrons que, pour tout "n" appartenant à "N" supérieur à 1, l'indice du sous-groupe "nZ" (que nous noterons "H") dans le groupe additif "(Z, +)".
Exprimons "nZ".
</p>
<math><mforall> b <min> Z, b <min> H <mequi> [<mexists> c <min> Z, b = c * n]</math>
<p>
Exprimons la classe à gauche de "a" suivant "H", pour tout "a" positif dans "Z".
</p>
<math><mforall>a <min> N, <mforall>d <min> Z, d <min> aH <mimp> [<mexists> e <min> nZ, d = a + e = a + c * n]</math>
<p>
Si "a" est inférieur à "n", alors on ne peut pas simplifier l'expression de "d".
Dans ce cas, pour tout "a" (inférieur à "n") différent, il existe un unique "d".
Donc, il existe au moins "n" classes dans l'ensemble des classes à gauche.
</p>
<math>a <mlt> n <mimp> [<mforall> f <min> N, aH = fH <mequi> a = f]</math>
<p>
Si "a" est supérieur ou égal à "n", on peut décomposer "c", pour revenir à une classe d'élément inférieure à "n".
Donc, on ne rajoute pas de nouvelles classes dans ce cas là.
Si cette décomposition permet de se ramener à un "a" (ou, dans l'exemple qui suit, "g") plus petit que "n", alors on se ramène au cas d'avaant.
Sinon, on recommence le procédé.
</p>
<math>a <mgt> n <mimp> [<mexists>g <min> N, <mexists> h <min> Z, a + c * n = (a - n) + (c + 1) * n = g + h * n] <mequi> [<mi>On recommence</mi> si g <mlt> n]</math>
<p>
Dans tous les cas où "a" appartient à "N", alors on obtient "n" classes différentes.
Si "a" est inférieur à "0", alors on applique le même principe, mais on ne soustrait pas "n" à "a", mais on l'ajoute.
Dans ce cas, on enlève "1" à "c".
Comme pour en haut, on ne crée pas de nouvelle classe.
Si ce nombre reste négatif, on réitère l'opération.
Sinon, on se retrouve dans le cas où "a" est plus grand que "n".
</p>
<math>a <mlt> 0 <mimp> [<mexists>g, h <min> Z, a + c * n = (a + n) + (c - 1) * n = g + h * n] <mequi> [<mi>On recommence</mi> si g <mlt> 0]</math>
<p>
Dans tous les cas, on se retrouve aveec "n" classes d'éléments suivant H.
Donc, l'ensemble des classes à gauche suivant "nZ" dans "(Z, +)" a pour cardinal "n".
</p>

<h4>2.0.1.3</h4>
<p>
Démontrons que le magma "(Z/nZ, +)" est un groupe abélien.
Déjà, vérifions que "Z/nZ" est stable par l'addition.
</p>
<math><mforall>a, b <min> Z/nZ, <mexists> c, d <min> N <mand> c <mlt> n <mand> d <mlt> n, <mexists> e, f <min> Z, a = c + e * n <mand> b = d + f * n</math></br>
<math>a + b = c + e * n + d + f * n = (c + d) + (e + f) * n</math>
<p>
Or, rien n'empêche "c + d" à être supérieur à "n".
Pour être sur d'éviter ce problème, comme on sait que "c + d" est inféérieur à "2 * n", alors on peut soustraite "n" à "c + d" si il est trop grand, et enlever "1" à "e + f" pour rééquilibrer le tout.
</p>
<math>c + d <mgtequal> n <mequi>((c + d) - n) + ((e + f) - 1) * n</math>
<p>
Dans tous les cas "c + d" (où "(c + d) - n" si nécessaire) est un nombre entier naturel inférieur à "n", et "e + f" ou "(e + f) - 1" est un nombre entier relatif.
Donc, le résultat de l'addition est un élément de "Z/nZ".
Donc, "Z/nZ" est stable par l'addition.
</p>
<math><mforall>a, b <min> Z/nZ, a + b <min> Z/nZ</math>
<p>
Maintenant, démontrons l'associativité de la loi.
Pour rappel, on peut exprimer la formule de l'addition, et utiliser des propriétés de "Z" et "N" pour facilement trouver le nécessaire.
</p>
<math><mforall>g <min> Z/nZ, <mexists>h <min> N <mand> h <mlt> n, <mexists>i <min> Z, g = h + i * n</math></br>
<math>(a + b) + g = (c + e * n + d + f * n) + h + i * n = c + e * n + (d + f * n + h + i * n) = a + (b + g)</math>
<p>
Donc, l'addition est bien associative sur "Z/nZ".
Maintenant, prouvons l'existence d'un neutre, qui est "[0]".
</p>
<math><mexists> j <min> Z, [0] = j * n</math>
<math><mforall>k <min> [0], a + k = c + e * n + j * n = c + (e + j) * n = a</math>
<p>
Donc, il y a bel et bien un neutre dans "Z/nZ".
Finalement, trouvons la présence d'un élément opposé pour toute valeur de "Z/nZ".
</p>
<math><mforall>l <min> Z/nZ, <mexists>m <min> N <mand> m <ml> n, <mexists>o <min> Z, l = m + o * n</math></br>
<math>a + l = [0] <mimp> c + e * n + m + o * n = c + m + (e + o) * n = [0] <mimp> c + m = 0 <mimp>c = -m = n - m</mimp></math>
<p>
Donc, chaque élément possède bel et bien un inverse.
Finalement, prouvons la commutativité.
Encore une fois, utilisons des propriétés de "Z" et "N" pour facilement trouver le nécessaire.
</p>
<math>a + b = c + e * n + d + f * n = d + f * n + c + e * n = b + a</math>
<p>
Donc l'addition est bel et bien commutative dans "Z/nZ".
Au final, "(Z/nZ, +)" est bel et bien un groupe abélien.
</p>


<h2>L'arithmétique des nombres rationnels</h2>

<h3 id="rational_addition">L'opération d'addition</h3>
<h4>2.1.0.0</h4>
<p>
Démontrons que l'addition de deux nombres rationnels "a" et "b" donne un nombre rationnel (et donc, que l'addition est stable dans "Q").
Soit deux nombres rationnels "a" et "b" :
</p>
<math><mforall>a, b <min> Q, <mexists>c, e <min> Z, <mexists>d, f <min> N, a = <mfrac><mi>c</mi><mi>d</mi></mfrac>, b = <mfrac><mi>e</mi><mi>f</mi></mfrac></math>
<p>
Admettons un objet "g", représentant "a + b".
Nous pouvons l'écrire avec la formule de l'addition de nombres rationnels.
</p>
<math>g = a + b = <mfrac><mi>(c * e) + (d * f)</mi><mi>e * f</mi></mfrac></math>
<p>
Or, comme "Z" est stable par l'addition et la multiplication, alors "(c * e) + (d * f)" appartient à "Z".
De plus, "N*" est stable par l'addition, donc "e * f" appartient à "N*".
Donc, "g" est un élément de "Z" sur un élément de "N*".
Donc, "g" est un nombre rationnel.
</p>

<h4>2.1.0.1</h4>
<p>
Démontrons que l'ordre d'addition de deux trois nombres rationnels "a", "b" et "c" n'importe pas (et donc, que l'addition est associative dans "Q").
Soit trois nombres rationnels "a", "b" et "c" :
</p>
<math><mforall>a, b, c <min> Q, <mexists>d, f, h <min> Z, <mexists>e, g, i <min> N, a = <mfrac><mi>d</mi><mi>e</mi></mfrac>, b = <mfrac><mi>f</mi><mi>g</mi></mfrac>, c = <mfrac><mi>h</mi><mi>i</mi></mfrac></math>
<p>
Admettons un objet "j", représentant "(a + b) + c".
Nous pouvons l'écrire avec la formule de l'addition de nombres rationnels.
</p>
<math>j = (a + b) + c = <mfrac><mi>((d * g) + (e * f)) * i + (e * g) * h</mi><mi>(e * g) * i</mi></mfrac></math>
<p>
En utilisant les propriétés calculatoire de "Z" et "N", réarrangeons "j".
</p>
<math>j = <mfrac><mi>((d * g) + (e * f)) * i + (e * g) * h</mi><mi>(e * g) * i</mi></mfrac></math></br>
<math>j = <mfrac><mi>(d * g) * i + (e * f) * i + (e * g) * h</mi><mi>(e * g) * i</mi></mfrac></math></br>
<math>j = <mfrac><mi>(d * g) * i + (e * f) * i + (e * g) * h</mi><mi>(e * g) * i</mi></mfrac></math></br>
<math>j = <mfrac><mi>d * (g * i) + e * ((f * i) + (g * h))</mi><mi>e * (g * i)</mi></mfrac></math></br>
<math>j = <mfrac><mi>d</mi><mi>e</mi></mfrac> + <mfrac><mi>(f * i) + (g * h)</mi><mi>g * i</mi></mfrac> = a + (b + c) </math>
<p>
Donc, "(a + b) + c" est égal à "a + (b + c)".
L'opération d'addition est associative dans "Q".
</p>
<math><mforall>a, b, c <min> Q, (a + b) + c = a + (b + c)</math>

<h4>2.1.0.2</h4>
<p>
Démontrons que l'ordre des opérandes pour l'addition entre deux nombres rationnels "a" et "b" n'importe pas (et donc, que l'addition est commutative dans "Q").
Soit deux nombres rationnels "a" et "b" :
</p>
<math><mforall>a, b <min> Q, <mexists>c, e <min> Z, <mexists>d, f <min> N, a = <mfrac><mi>c</mi><mi>d</mi></mfrac>, b = <mfrac><mi>e</mi><mi>f</mi></mfrac></math>
<p>
Admettons un objet "g", représentant "a + b".
Nous pouvons l'écrire avec la formule de l'addition de nombres rationnels.
</p>
<math>g = a + b = <mfrac><mi>(c * e) + (d * f)</mi><mi>e * f</mi></mfrac></math>
<p>
En utilisant les propriétés calculatoire de "Z" et "N", réarrangeons "j".
</p>
<math>g = <mfrac><mi>(c * e) + (d * f)</mi><mi>e * f</mi></mfrac> = <mfrac><mi>(d * f) + (c * e)</mi><mi>f * e</mi></mfrac> = b + a</math>
<p>
Donc, "a + b" est égal à "b + a".
Donc, l'addition est commutative dans "Q".
</p>
<math><mforall>a, b <min> Q, a + b = b + a</math>

<h4>2.1.0.3</h4>
<p>
Démontrons que la structure algébrique formée par l'addition et "Q" est un groupe abélien.
Déjà, selon le théorème 2.1.0.0, "+" est stable dans "Q", alors l'addition dans "Q" forme bien une structure algébrique.
Selon le théorème 2.1.0.1, "+" est associative dans "Q".
Selon le théorème 2.1.0.2, "+" est commutative dans "Q".
De plus, les fractions ayant "0" comme numérateur se comportent comme des éléments neutres.
Donc, cette classe d'équivalence dans "Q" représente l'élément neutre de "Q".
</p>
<math><mforall>a, b <min> Q, <mexists>c <min> Z, <mexists>d, e <min> N, a = <mfrac><mi>c</mi><mi>d</mi></mfrac>, b = <mfrac><mi>0</mi><mi>e</mi></mfrac></math></br>
<math>a + b = <mfrac><mi>c * e + 0 * d</mi><mi>d * e</mi></mfrac> = <mfrac><mi>c</mi><mi>e</mi></mfrac> = a</math>
<p>
Finalement, chaque élément possède un inverse, donnant le neutre quand on les additionne.
</p>
<math><mforall>a <min> Q, <mexists> b <min> Q, <mexists>c <min> Z, <mexists>d <min> N, a = <mfrac><mi>c</mi><mi>d</mi></mfrac>, b = <mfrac><mi>-c</mi><mi>d</mi></mfrac></math></br>
<math>a + b = <mfrac><mi>c * d + -c * d</mi><mi>d * d</mi></mfrac> = <mfrac><mi>0</mi><mi>d * d</mi></mfrac> = 0</math>
<p>
Donc, la structure algébrique "(Q, +)" est un groupe abélien.
</p>

<h3 id="rational_multiplication">L'opération de multiplication</h3>
<h4>2.1.1.0</h4>
<p>
Démontrons que la multiplication de deux nombres rationnels "a" et "b" donne un nombre rationnel (et donc, que la multiplication est stable dans "Q").
Soit deux nombres rationnels "a" et "b" :
</p>
<math><mforall>a, b <min> Q, <mexists>c, e <min> Z, <mexists>d, f <min> N, a = <mfrac><mi>c</mi><mi>d</mi></mfrac>, b = <mfrac><mi>e</mi><mi>f</mi></mfrac></math>
<p>
Admettons un objet "g", représentant "a * b".
Nous pouvons l'écrire avec la formule de l'addition de nombres rationnels.
</p>
<math>g = a + b = <mfrac><mi>c * e</mi><mi>d * f</mi></mfrac></math>
<p>
Or, comme "Z" est stable par la multiplication, alors "c * e" appartient à "Z".
De plus, "N*" est stable par la multiplication, donc "d * f" appartient à "N*".
Donc, "g" est un élément de "Z" sur un élément de "N*".
Donc, "g" est un nombre rationnel.
</p>

<h4>2.1.1.1</h4>
<p>
Démontrons que l'ordre de la multiplication de deux trois nombres rationnels "a", "b" et "c" n'importe pas (et donc, que la multiplication est associative dans "Q").
Soit trois nombres rationnels "a", "b" et "c" :
</p>
<math><mforall>a, b, c <min> Q, <mexists>d, f, h <min> Z, <mexists>e, g, i <min> N, a = <mfrac><mi>d</mi><mi>e</mi></mfrac>, b = <mfrac><mi>f</mi><mi>g</mi></mfrac>, c = <mfrac><mi>h</mi><mi>i</mi></mfrac></math>
<p>
Admettons un objet "j", représentant "(a * b) * c".
Nous pouvons l'écrire avec la formule de l'addition de nombres rationnels.
</p>
<math>j = (a * b) * c = <mfrac><mi>(d * f) * h</mi><mi>(e * g) * i</mi></mfrac></math>
<p>
En utilisant les propriétés calculatoire de "Z" et "N", réarrangeons "j".
</p>
<math>j = <mfrac><mi>(d * f) * h</mi><mi>(e * g) * i</mi></mfrac></math></br>
<math>j = <mfrac><mi>d * (f * h)</mi><mi>e * (g * i)</mi></mfrac> = a * (b * c)</math></br>
<p>
Donc, "(a * b) * c" est égal à "a * (b * c)".
L'opération de multiplication est associative dans "Q".
</p>
<math><mforall>a, b, c <min> Q, (a * b) * c = a * (b * c)</math>

<h4>2.1.1.2</h4>
<p>
Démontrons que l'ordre des opérandes pour la multiplication entre deux nombres rationnels "a" et "b" n'importe pas (et donc, que la multiplication est commutative dans "Q").
Soit deux nombres rationnels "a" et "b" :
</p>
<math><mforall>a, b <min> Q, <mexists>c, e <min> Z, <mexists>d, f <min> N, a = <mfrac><mi>c</mi><mi>d</mi></mfrac>, b = <mfrac><mi>e</mi><mi>f</mi></mfrac></math>
<p>
Admettons un objet "g", représentant "a * b".
Nous pouvons l'écrire avec la formule de l'addition de nombres rationnels.
</p>
<math>g = a * b = <mfrac><mi>c * e</mi><mi>d * f</mi></mfrac></math>
<p>
En utilisant les propriétés calculatoire de "Z" et "N", réarrangeons "j".
</p>
<math>g = <mfrac><mi>c * e</mi><mi>d * f</mi></mfrac> = <mfrac><mi>e * c</mi><mi>f * d</mi></mfrac> = b * a</math>
<p>
Donc, "a * b" est égal à "b * a".
Donc, la multiplication est commutative dans "Q".
</p>
<math><mforall>a, b <min> Q, a * b = b * a</math>

<h4>2.1.1.3</h4>
<p>
Démontrons que la structure algébrique formée par la multiplication et "Q*" est un groupe abélien.
Déjà, selon le théorème 2.1.1.0, "*" est stable dans "Q*", alors l'addition dans "Q*" forme bien une structure algébrique.
Selon le théorème 2.1.1.1, "*" est associative dans "Q".
Selon le théorème 2.1.1.2, "*" est commutative dans "Q".
De plus, les fractions valant 1 se comportent comme des éléments neutres.
Donc, cette classe d'équivalence dans "Q" représente l'élément neutre de "Q".
</p>
<math><mforall>a, b <min> Q, <mexists>c <min> Z, <mexists> e <min> Z, <mexists>d, <min> N, a = <mfrac><mi>c</mi><mi>d</mi></mfrac>, b = <mfrac><mi>e</mi><mi>e</mi></mfrac></math></br>
<math>a * b = <mfrac><mi>c * e</mi><mi>d * e</mi></mfrac> = <mfrac><mi>c</mi><mi>d</mi></mfrac> = a</math>
<p>
Finalement, chaque élément possède un inverse, donnant le neutre quand on les multiplient.
</p>
<math><mforall>a <min> Q, <mexists> b <min> Q, <mexists>c <min> Z, <mexists>d <min> N, a = <mfrac><mi>c</mi><mi>d</mi></mfrac>, b = <mfrac><mi>d</mi><mi>c</mi></mfrac></math></br>
<math>a * b = <mfrac><mi>c * d</mi><mi>d * c</mi></mfrac> = <mfrac><mi>c * d</mi><mi>c * d</mi></mfrac> = 1</math>
<p>
Donc, la structure algébrique "(Q*, *)" est un groupe abélien.
</p>

<h4>2.1.1.4</h4>
<p>
Démontrons que dire que, pour tout nombre rationnel a, b et c "a = b" équivaut à dire que "a * c = b * c".
Soit trois nombres rationnels "a", "b" et "c" :
</p>
<math><mforall>a, b, c <min> Q, <mexists>c, e, g <min> Z, <mexists>d, f, h <min> N, a = <mfrac><mi>c</mi><mi>d</mi></mfrac>, b = <mfrac><mi>e</mi><mi>f</mi></mfrac>, c = <mfrac><mi>g</mi><mi>h</mi></mfrac></math>
<p>
Admettons que "a = b".
</p>
<math>a = b <mequi> c * f = d * e</math>
<p>
En utilisant les propriétés calculatoire de "Z" et "N", calculons des deux côtés par "g" et "h".
</p>
<math>a = b <mequi> c * f = d * e <mequi> c * f * g * h = d * e * g * h <mequi> (c * g) * (f * h) = (e * g) * (d * h) <mequi> a * c = b * c</math>
<p>
Donc, "a = b" est équivalent à dire que "a * c = b * c".
</p>
<math><mforall>a, b, c <min> Q, a = b <mequi> a * c = b * c</math>
